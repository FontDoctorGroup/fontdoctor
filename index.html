Abstract

汉字字体不仅是中华文化传承的载体，更是东方美学的视觉呈现。虽然社会对多样化字体的渴望空前高涨，但现有的制作工艺难以逾越海量字符绘制与复杂协同流程带来的挑战，特别是人工质检固有的低效性。针对汉字字体质量检查的需求，本文提出FontDoctor，一款对汉字字体缺陷高精度定位的字体缺陷检测多模态大模型。FontDoctor 具有few-shot template-referenced 和 non-referenced universal 两类互补的汉字字体缺陷检测范式。为了将缺陷检测任务有效整合进 MLLM，我们利用 SVG XML 在汉字字形光栅图像与矢量数据间建立语义映射，使模型能够通过理解SVG 实现精确地定位并标记字形缺陷。针对多模板带来的上下文瓶颈，我们将离散 SVG XML 字符流转化为稠密视觉嵌入，使MLLM视觉编码器兼具“语义理解”与“数据压缩”的双重功能，实现极具紧凑性的特征表示。为推动字体缺陷检测技术发展，我们提出一个包含两百万个字形大规模字体缺数据集CFDefect-2M。据我们所知，本文是首个专注于汉字字体缺陷检测的工作。大量实验表明，Font Docto不仅实现了最先进性能，更展现出融入专业汉字字体设计工作流的巨大潜力。
